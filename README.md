# 🌦️ Weather App

## 📌 Project Description
This is a weather web application that allows users to search for any city and view real-time weather information such as temperature, humidity, and weather conditions.

## 🔌 API Used
OpenWeather API is used to fetch real-time weather data.

## 🚀 Features (Planned)
- Search weather by city name
- Display temperature, humidity, and conditions
- Responsive design for all devices
- Filter and sort features (to be implemented)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## ▶️ How to Run
1. Clone the repository
2. Open index.html in your browser
